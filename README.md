## 🎯 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/GPU_Programming.git
   cd GPU_Programming
   ```

2. **Navigate to a challenge**
   ```bash
   cd challenges/vector_addition
   ```

3. **Compile and run**
   ```bash
   nvcc -o program main.cu
   ./program
   ```
